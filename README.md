# YouTube Transcript Downloader

Download YouTube transcripts and send them to Claude with one click!

## Features
- Download transcripts from YouTube videos
- Send transcripts directly to Claude for summarization

## Installation
1. Download this repository as ZIP
2. Extract the files
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode"
5. Click "Load unpacked" and select the extension folder

## Usage
- Click "📥 Download" to save transcript as .txt file
- Click "📋 Copy to Claude" to send transcript to Claude
